# 🔐 Smart Door System (Arduino)

## 📌 Overview
This project is a smart door system using Arduino with:
- Keypad for password input
- Servo motor for door control
- Buzzer for wrong password alert

## ⚙️ Features
- Enter password via keypad
- Correct password → door opens (servo rotates)
- Wrong password → buzzer alarm

## 🛠️ Hardware
- Arduino Uno
- Keypad 4x4
- Servo motor (SG90)
- Buzzer

## 🔌 Pin Connection
- Servo: pin 11
- Buzzer: pin 12
- Keypad: pins 2–9

## 📁 Code
Main file: `smart_door.ino`
